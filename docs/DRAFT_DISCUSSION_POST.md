### Docker Deployment Now Available

Based on feedback, I've added **one-command Docker deployment**. No more installing Python, Node, SQL Server separately.

```bash
git pull                    # Get latest
docker-compose up -d        # Start everything
# Open http://localhost:5173
```

That's it. Database, backend, frontend - all containerized.

See the new [INSTALL.md](../INSTALL.md) for full instructions.
